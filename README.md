# Crypto Currency

## ➡️ [Live Demo](https://crypto-currency-nc.netlify.app/)

## :key: Get API Key 

- [https://www.coingecko.com/en/api](https://www.coingecko.com/en/api)

## :computer: Setup

1. Clone the repository and install its features.

```
yarn install
```

2. Use the project locally (localhost).

```
yarn start
```
